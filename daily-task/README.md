# DailyTask - 每日任务管理器

一个简洁的桌面任务管理应用，支持任务制定、时间提醒、历史记录查看等功能。

## 功能特性

- 制定每日任务和时间节点
- 完成任务手动打勾
- 任务到达时间节点时系统弹窗提醒
- 查看历史每天的任务完成情况
- 多种主题模式（浅色/深色/高对比度/复古色）
- 窗口置顶功能（简洁模式）
- 边缘隐藏功能（类似QQ）
- 系统托盘运行

## 技术栈

- Tauri 2.x + Vanilla JavaScript
- Rust 后端
- 数据存储：本地 JSON 文件

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建发布
npm run tauri build
```

## 快捷操作

- 点击图钉按钮：置顶窗口（简洁模式）
- 点击 ◐ 按钮：切换主题
- 点击 ⊞ 按钮：边缘隐藏（循环切换左/右/顶部）
