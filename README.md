# DailyTask 每日任务管理器

[English](#english)

---

## 功能一览

| 功能 | 说明 |
|------|------|
| 📝 任务管理 | 添加任务名称和截止时间 |
| ✅ 完成标记 | 点击复选框完成/取消任务 |
| 🔔 时间提醒 | 任务到达截止时间自动弹窗提醒 |
| 📅 历史记录 | 查看每日任务完成情况 |
| 🎨 主题切换 | 浅色/深色/高对比度/复古色 |
| 📌 置顶模式 | 窗口置顶，简洁显示任务 |
| 🫥 边缘隐藏 | 类似QQ的边缘隐藏功能 |
| 📥 系统托盘 | 最小化到托盘运行 |

---

## 快速开始

```bash
# 克隆项目
git clone <repo-url>
cd daily-task

# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建发布
npm run tauri build
```

## English

# DailyTask - Desktop Task Manager

## Features

- Task planning with deadline
- Checkbox to mark completion
- System notification on deadline
- History tracking
- 4 themes: Light/Dark/High Contrast/Sepia
- Always-on-top compact mode
- Edge hiding (like QQ)
- System tray support

## Quick Start

```bash
git clone <repo-url>
cd daily-task
npm install
npm run tauri dev
npm run tauri build
```

## Tech Stack

- Tauri 2.x + Rust
- Vanilla JavaScript
- JSON local storage
